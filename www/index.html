RLSD

Overview

Welcome! You've reached the home of RLSD, the Retro Linux-libre Software Distribution.

RLSD is an extremely minimalistic, small, lightweight and security-hardened, free, text operating system for x86 and x86_64. It's a unique GNU/Linux-libre distribution that provides a selection of console applications and home-grown security features which might appeal to hackers.

In RLSD:

  • The user always runs as root
  • The root user is called "boss"
  • Everything is linked statically and the kernel is 100% monolithic
  • The file system layout is extremely simple: all executables go to /bin, man pages to /man, etc'
  • Package management is simple and fast, thanks to packlim
  • Packages are small; for example, the C library is musl and the shell is loksh
  • luufs prevents malware from tampering with the operating system core
  • logfence prevents modification of logs
  • elfence prevents the execution of untrusted applications
  • A kernel security module ensures executables run only from /bin and blocks malware-like activity (e.g mprotect() with PROT_EXEC)
  • The user is trapped in a container, to protect critical system daemons
  • Critical daemons run in a private mounts namespace, to prevent mount-based attacks
  • The operating system core is a digitally-signed image
  • All security features can be disabled by passing "secure=0" to the kernel, but that requires reboot
  • Copy-paste using the mouse works and handled by gpm
  • The console color scheme is randomly generated at boot time

Downloads

  • ISO images, with support for dd
  • Binary packages, via the repository
  • Source code for all packages
  • Build logs for all packages

Usage

  • Use ifconfig, dhcpcd, iw, wpa_passhprase and wpa_supplicant to set up a network connection
  • Use packlim to install packages
  • Use man to obtain documentation

Development

Most of RLSD's development is automated, to ease its development; for example, package updates are monitored here and the entire package repository is built automatically. To build your own version of RLSD or contribute, clone its code repository and follow the documentation.

If you encounter a crash, please send the crashing executable from /bin and the crash dump from /run/crash. Thank you.
